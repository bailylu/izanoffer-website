# 0002 · 品牌名定为 izanoffer

- **日期**: 2026-06-25
- **状态**: 已采纳
- **决策人**: Joe / 共同

---

## 背景

整个产品方案 v0.3 阶段一直用「留学 AI」作为占位名。建立 GitHub 仓库时需要正式品牌名，进一步推进设计、品牌识别、对外宣传也都需要一个真实名字。

## 候选方案

### 方案 A · 留学 AI（继续用占位名）
- 描述：保留中文描述性名字
- 优点：直白、用户一秒懂
- 缺点：太通用、不可注册、无品牌识别度、SEO 难做

### 方案 B · izanoffer
- 描述：自造词，"iz an offer" 谐音「拿一个 offer」/「Easy an offer」
- 优点：原创可注册、暗含目标（拿 offer）、域名好抢、全小写有现代科技感
- 缺点：中文用户首次见可能不会念

### 方案 C · 其他候选
- 留学魔方 / 启航 / Voyager / Atlas / Aboard 等
- 优点：各有寓意
- 缺点：要么被占用，要么不够独特

## 决定

采用 **izanoffer**，全小写写法（参考 vercel / supabase / linear 的现代品牌风格）。

## 理由

1. **原创可注册** —— 自造词，几乎不会和已有商标冲突，便于工商注册和域名抢注。
2. **目标导向** —— 名字直接指向用户的核心诉求（拿 offer）。学生看到名字就知道这个产品是帮我什么的。
3. **现代科技品牌感** —— 全小写、单词式（无连字符）、可读性强，符合当代 SaaS / AI 产品的命名风格。
4. **跨语言一致** —— 中英文场合都用一个名字，不需要两套品牌系统。

## 后果

**正面**：
- 可以立即注册商标、买域名（izanoffer.com / .ai / .cn）
- 可以开始设计 Logo / VI / Voice
- 文档、原型、对外材料统一品牌

**负面 / 取舍**：
- 首次接触的中国用户可能不会念（建议在 logo 旁边偶尔标注音译，比如「Izan Offer · 易赞欧法」之类，待品牌策略阶段定）
- 国内搜索引擎 SEO 一开始会弱，需要持续做品牌词内容沉淀

**需要立刻做的事**：
- [x] 项目文档 / README / 原型 全局替换"留学 AI" → "izanoffer"
- [ ] 抢注域名：izanoffer.com / izanoffer.ai / izanoffer.cn（优先级最高）
- [ ] 抢注小红书 / 微信公众号 / 抖音 / 即刻 等社交账号

## 后续

什么情况下应该回来重新审视？

- 域名 / 商标抢不到 → 必须换名（紧急）
- 用户反馈名字记不住、传不开 → 6 个月后看数据再说
